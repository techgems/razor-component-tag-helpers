# Deprecation notice

As of February 2nd, 2026. This package has been deprecated and this repository will no longer be updated. Further development will continue in [Static Components](https://github.com/techgems/static-components) with upcoming updates to more easily work with AlpineJS, HTMX and a better documentation as well as guides on how to create reusable component libraries in the form of Razor Class Libraries.

- [New documentation](https://static-components.techgems.net/)
- [Nuget Package](https://www.nuget.org/packages/TechGems.StaticComponents)
 
## Old package and documentation

Razor Component Tag Helpers is a one file ASP.NET Core library that allows you to write UI components while maintaining compatibility with Razor Pages and MVC.

- [Documentation](https://razor-components.techgems.net/)
- [Nuget Package](https://www.nuget.org/packages/TechGems.RazorComponentTagHelpers)
